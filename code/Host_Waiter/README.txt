\f0\fs24 \cf0 ## READ ME - Host/Waiter interface\
\
*** Before you begin make sure have the Ionic CLI and nodemon CLI downloaded.\
\
CLIENT\
++++++++\
1. Open the client folder directory on your terminal.\
2. On the terminal, type < npm install > - this will install of the modules inside the package.json file.\
3. On the terminal, < npm start > - this will open the webpage on your default browser.\
(The client should be on port 3000)\
\
SERVER\
++++++++\
1. Open the server folder directory on your terminal.\
2. On the terminal, type < npm install >.\
3. Make sure you have < nodemon > installed.  \
	> To install nodemon - On the terminal, type < npm install -g nodemon >.\
4. On the terminal, type < nodemon > to start the server.\
(The server should be on port 4000)\
\
MOBILE\
++++++++\
1. Open the mobile folder directory on your terminal.\
2. On the terminal, type < npm install >.\
3. Make sure that you have the Ionic framework CLI downloaded before ups continue.\
4. Run the mobile application:\
	> For testing purposes (to run the application on a browser) - On the terminal 		type, < ionic serve >.\
	> To upload to an ionic application (to put the application on a phone) - On the 	terminal, type < ionic upload > .\
	> To compile down to Java or C - On the terminal type, < ionic build ios >(for 		iOS) or < ionic build android > (for an Android application)\
